Movie Recommendation System
🎬 Project Title

Content-Based Movie Recommendation System

📌 Overview

This project builds a Content-Based Recommendation System using movie data.
It recommends movies based on similarity between items (movies) using features like genres.

Instead of recommending based on users, this system finds similar movies to a given movie.

📂 Dataset Used

The project uses the following datasets:

movies.csv → Contains movie details (movieId, title, genres)
ratings.csv → User ratings for movies
users.csv → User information (optional use)
